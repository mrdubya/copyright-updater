Test file for Copyright notice formats that can be updated.

NOTE: This file uses UTF-8 encoding.

Procedure:
1. Load this file.
2. Modify the file in some manner.
3. Add Fred Bloggs for updates with :CopyrUpdateFor 'Fred Bloggs'
4. Save the file - check all but last block update.
